# Базові структури даних у Python

Python пропонує декілька потужних вбудованих структур даних для організації інформації.

## Ключові структури:

1.  **Список (List):**
    * Упорядкована, змінна (mutable) колекція елементів.
    * Створюється за допомогою квадратних дужок: `my_list = [1, "hello", 3.14]`.
2.  **Кортеж (Tuple):**
    * Упорядкована, незмінна (immutable) колекція елементів.
    * Створюється за допомогою круглих дужок: `my_tuple = (1, 2, 3)`.
3.  **Множина (Set):**
    * Невпорядкована колекція унікальних елементів. Видаляє дублікати.
    * Створюється за допомогою фігурних дужок або функції `set()`: `my_set = {1, 2, 3}`.
4.  **Словник (Dictionary):**
    * Невпорядкована колекція пар **ключ:значення**.
    * Ключі мають бути унікальними та незмінними.
    * Створюється за допомогою фігурних дужок: `my_dict = {"name": "Alice", "age": 30}`.

# Доступ до елементів словника
