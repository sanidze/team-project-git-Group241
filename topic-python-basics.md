# Основи мови програмування Python

Python — це високорівнева, інтерпретована мова програмування загального призначення.

## Основні поняття:

1.  **Змінні та типи даних:**
    * Python є динамічно типізованою мовою, тобто тип змінної визначається під час виконання.
    * Основні типи: `int` (цілі числа), `float` (дробові числа), `str` (рядки), `bool` (булеві значення).
2.  **Оператори:**
    * Арифметичні (`+`, `-`, `*`, `/`, `%`).
    * Порівняння (`==`, `!=`, `<`, `>`).
    * Логічні (`and`, `or`, `not`).
3.  **Умовні конструкції:**
    * Використовується синтаксис `if`, `elif`, `else` для розгалуження коду.
    * **Важливо:** Python використовує відступи (індексацію) для визначення блоків коду, а не фігурні дужки.

## Приклад:

```python
# Приклад умовного оператора

if age > 18:
    print("Дорослий")
else:
